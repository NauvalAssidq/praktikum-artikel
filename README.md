# Project Progress Overview

Dokumentasi ringkas mengenai progres pengerjaan project berdasarkan minggu ke-2 dan minggu ke-3.

---

## Minggu 2  
### **Pembuatan Migration, Model Sederhana, dan AuthController**

Pada minggu kedua, fokus pengembangan berada pada pondasi awal aplikasi, meliputi:

- **Pembuatan Migration**
  - Membuat struktur tabel dasar yang dibutuhkan aplikasi.
  - Menentukan field penting seperti ID, timestamps, dan atribut sesuai kebutuhan awal.

- **Pembuatan Model Sederhana**
  - Membuat model untuk berkomunikasi dengan tabel database yang sudah dibuat.
  - Menambahkan properti dasar seperti `$fillable` atau `$guarded`.

- **Pembuatan AuthController**
  - Menambahkan dua function utama:
    - `loginIndex()` – Menampilkan halaman login.
    - `registerIndex()` – Menampilkan halaman registrasi.
  - Menyiapkan struktur dasar untuk auth flow aplikasi.

---

## Minggu 3  
### **Pengecekan Running Project & Validasi Item Minggu 2**

Pada minggu ketiga, fokus utama adalah memastikan semua fitur dasar yang dibuat pada minggu sebelumnya berjalan dengan baik:

- **Pengecekan Running Project**
  - Melakukan pengecekan apakah project dapat berjalan tanpa error.
  - Testing migration, model, dan routing dasar.

- **Validasi Item Minggu 2**
  - Mengecek kembali migration, struktur model, dan function pada AuthController.
  - Memastikan fungsi loginIndex dan registerIndex dapat diakses dengan benar.

## Status
